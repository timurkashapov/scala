Переменные (variable) Переменные состоят из 2 типов:
  • Изменяемые (mutable) - объявляются зарезервированным, ключевым словом var
  • Неизменяемые (immutable) - объявляются зарезервированным, ключевым словом val
Синтаксис объявления переменной: var num:Int = 4
ключевое слово идентификатор знак принадлежности тип переменной оператор присваивания целочисленное значение
var num : Int = 4
Временная переменная (temporary variable) - создаются by Scala, когда не задаются исходным переменным идентификаторы (имена)
